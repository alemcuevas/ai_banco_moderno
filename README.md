# ◆ NeoBank — Banco Digital Moderno

> *Tu dinero, sin límites.*

NeoBank es una landing page moderna para un banco 100% digital, construida con HTML y CSS puro. Presenta un diseño oscuro elegante con acentos en púrpura y cyan, animaciones fluidas y una experiencia completamente responsive.

## 🎨 Diseño y Características

- **Dark Theme** — Fondo oscuro (#0a0a1a) con acentos vibrantes
- **Gradientes** — Púrpura → lavanda → cyan como identidad visual
- **Glassmorphism** — Navbar con `backdrop-filter: blur(20px)`
- **Tarjeta 3D** — Perspectiva CSS con hover interactivo
- **Animaciones** — Formas flotantes, notificaciones con `@keyframes`
- **Tipografía** — Inter (Google Fonts) con pesos de 300 a 800
- **CSS Variables** — Sistema de diseño completo con custom properties
- **Responsive** — Breakpoints en 1024px, 768px y 480px

## 📱 Responsive Design

La página se adapta a todos los tamaños de pantalla:

- **Desktop** (>1024px) — Layout completo con grids de 3 columnas
- **Tablet** (768px–1024px) — Grids de 2 columnas, ajustes de tipografía
- **Móvil** (<768px) — Layout de 1 columna, menú hamburguesa, tarjeta adaptada
